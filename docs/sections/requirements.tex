\subsection{User Roles}

The platform defines three user roles with escalating privileges:

\begin{enumerate}[label=\textbf{\arabic*.}]
    \item \textbf{Regular User (Job Seeker)} — Can manage profile, upload resumes, apply to jobs, send/receive messages.
    \item \textbf{Recruiter / Company Admin} — All user capabilities plus creating company pages, posting jobs, viewing applicants, and messaging candidates.
    \item \textbf{Platform Admin} — Full access to user management, content moderation, and audit logs.
\end{enumerate}

Access control is enforced at the API layer via dependency injection (FastAPI \texttt{Depends()}) — no endpoint can be accessed without passing through the role checker.
