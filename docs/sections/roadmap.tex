\subsection{Bonus Features Under Consideration}

\begin{itemize}
    \item \textbf{Blockchain-based audit logging (+6\%):} Tamper-evident log chain where each entry contains the hash of the previous entry, making retroactive modification detectable.
    \item \textbf{Resume parsing and intelligent matching (+2\%):} NLP-based extraction of skills, experience, and education from uploaded PDFs to enable automated candidate–job matching.
\end{itemize}
