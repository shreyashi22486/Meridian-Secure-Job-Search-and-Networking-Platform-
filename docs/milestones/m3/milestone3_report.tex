\section{Overview}

This report covers Milestone~3 of the Meridian platform. Building on the authentication, profile management, encrypted resume storage, and admin dashboard delivered in M2, this milestone implements the core job-portal workflows and advanced security infrastructure:

\begin{enumerate}
    \item \textbf{Company pages and job postings} — Multi-level company RBAC (Owner/Admin/Editor), full CRUD for companies and jobs, with audit logging on every mutation.
    \item \textbf{Job search and application workflow} — Parameterized multi-filter search using JSONB operators, application submission with resume attachment and ownership verification, duplicate prevention via database constraints.
    \item \textbf{Application status tracking} — Five-state lifecycle (Applied → Reviewed → Interviewed → Offer/Rejected), strict data segregation between applicant and recruiter views, recruiters' private notes invisible to candidates.
    \item \textbf{Encrypted one-to-one and group messaging} — Client-side E2EE (AES-GCM) for direct messages with zero-knowledge server, server-side Fernet encryption for groups, PKI digital signatures on all messages, read receipts and unread counts.
    \item \textbf{Tamper-evident admin audit logging} — SHA-256 hash-chained audit entries with RSA-2048 PKI digital signatures, automated chain verification endpoint, OTP-gated destructive admin actions.
\end{enumerate}
